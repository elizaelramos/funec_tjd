## 🔑 Pontos-Chave do Fluxo

### As 7 Situações

```
1️⃣  AGUARDANDO CITAÇÃO
2️⃣  AGUARDANDO AGENDAMENTO  
3️⃣  AGENDADO
4️⃣  JULGADO - PERÍODO DE RECURSO (⚠️ 72 HORAS)
5️⃣  JULGADO (FINAL)
6️⃣  RECURSO ACEITO (se houver)
7️⃣  ARQUIVADO
```

### Prazo Crítico: 72 Horas para Recurso
```
Quando pauta é julgada:
  ↓
Sistema inicia contagem de 72 horas
  ↓
Se houver recurso → Situação muda para "Recurso Aceito"
Se não houver     → Situação muda para "Julgado" (final)
```

### Documentos Obrigatórios
- **Origem:** Denúncia/Petição inicial (criação)
- **Citação:** Comprovante de notificação (antes do julgamento)
- **Decisão:** Resultado do julgamento (após julgamento)
- **Recurso:** Petição (durante 72 horas)
